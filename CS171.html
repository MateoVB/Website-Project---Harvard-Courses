CS171 - Visualization


Instructor: Hanspeter Pfister

2019 Fall    Full Term    MW    3:00pm - 5:45pm

Description:
An introduction to key design principles and techniques for visualizing data. Covers design practices, data and image models, visual perception, interaction principles, visualization tools, and applications. Introduces programming of web-based interactive visualizations.

Course Notes:
Enrollment limited to 80 students.

Recommended Prep:
Students are required to have basic programming experience (e.g., Computer Science 50). Web programming experience (HTML, CSS, JS) is a plus.
